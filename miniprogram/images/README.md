# 图标文件说明

这个文件夹需要包含以下图标文件（建议尺寸 64x64px）：

## 底部导航栏图标
- `home.png` - 学习中心图标（未选中）
- `home-active.png` - 学习中心图标（选中）
- `topics.png` - 主题库图标（未选中）
- `topics-active.png` - 主题库图标（选中）
- `history.png` - 学习历史图标（未选中）
- `history-active.png` - 学习历史图标（选中）
- `settings.png` - 设置图标（未选中）
- `settings-active.png` - 设置图标（选中）
